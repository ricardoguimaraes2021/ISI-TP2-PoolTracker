# PoolTracker Web

Frontend React para o sistema PoolTracker.

## Configuração

### Variáveis de Ambiente

Crie um ficheiro `.env` na raiz do projeto com a seguinte configuração:

```
VITE_API_URL=http://localhost:5011
```

**Nota:** Certifique-se de que o servidor backend está a correr na porta 5011 antes de iniciar o frontend.

## Instalação

```bash
npm install
```

## Executar em Desenvolvimento

```bash
npm run dev
```

O frontend estará disponível em `http://localhost:5173` (ou outra porta se 5173 estiver ocupada).

## Build para Produção

```bash
npm run build
```
